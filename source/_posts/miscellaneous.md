# 关于 OI

初学 OI 的时候，纠结用 0-base 下标还是 1-base 下标，因为不想浪费下标为 0 的那一位。

后来发现，使用 1-base 下标的时候，下标 0 并没有被浪费。很多时候我们用下标 0 表示超级源点/dp 初始状态/trie 树的无效位/...，如果不使用下标 0，就需要特判之类的手动处理。并且在 0-base 和 1-base 之间换算也是相当烦人的，很多时候都会忘记究竟是 0-base 还是 1-base。

综上，处于方便，最终统一为了 1-base 下标。
